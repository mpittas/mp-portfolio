The "HELLO! My name is" app allows users to create personalized name tags using a variety of graffiti-inspired fonts. This web-based tool offers customization options for font selection, size, color, and effects such as rotation and letter spacing. Users can easily design their name tags by adjusting these settings and can see their changes in real time. The application aims to provide a fun, creative platform for users to generate unique, stylish name tags for personal or professional use.

**Key Features:**

- Multiple graffiti-style fonts
- Customizable font size, color, and effects
- Real-time design updates
- User-friendly interface for easy navigation and design adjustments
